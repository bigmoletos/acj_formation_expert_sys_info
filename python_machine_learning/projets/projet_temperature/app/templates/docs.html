Documentation de l'API

Introduction

Bienvenue dans la documentation de l'API Weather App. Cette application permet de :

  • Consulter la météo en temps réel
  • Gérer les utilisateurs
  • Sauvegarder les recherches favorites

Points d'accès API

GET /weather

Récupère les données météorologiques pour une ville donnée

Paramètres

  • city - Nom de la ville (requis)

POST /search/save

Sauvegarde une recherche dans les favoris

Corps de la requête

  • city - Nom de la ville
  • user_id - ID de l'utilisateur

Authentification

L'API utilise l'authentification par token JWT. Pour obtenir un token :

  1. Connectez-vous via /login
  2. Utilisez le token reçu dans le header Authorization
